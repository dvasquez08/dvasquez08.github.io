Works

Student Attendance App

Flutter

An attendance app that uses the device's camera as a barcode scanner. The app scans the barcode, then sends the data to a Firestore database and also captures the timestamp of the scan to track attendance. The names previously written down by hand. This app prevents human error and also disagreements between parents and staff when trying to confirm if they or their child did indeed attend. This tracks that they attended as well as the timestamp of when they attended. Check out the code here.

Student Mannager Web App

Flutter

This app features authentication, state management using Riverpod with the MVVM design pattern and stream builders to display the data from Firestore. At the time of the screenshots the app was connected to a test Firestore collection hence the names such as"student one." This app is still a work-in progress with more features planning to be built into it in the future., However, the core functions of it was a way for the business owner to view the stored data securely and that has now been complete. You can see the source code here.

SPTKD

Flutter

Public communication application for parents and students at a Taekwondo school. Resolved communication issues between staff and students and parents as there is a single location for notification and information where as paper flyers or verbal announcements were made to make announcements. This also minimized call volume as people could find most of the answers to their questions in the app.
